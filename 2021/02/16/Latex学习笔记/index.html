第三方差异高亮工具git-latexdiff的安装与使用

下载链接：https://gitlab.com/git-latexdiff/git-latexdiff

需要注意的点是：

On Windows

Windows users are likely to be using this tool inside the so-called “Git Bash for Windows” environment which is included in Git for Windows.

In this environment, the provided version of perl might be unable to run latexdiff. You can install a different perl distribution, for instance Strawberry Perl. Then, you must put Strawberry Perl’s folders before anything else in your PATH and run latexdiff and git-latexdiff.

Such a change could interfere with the tools in the “Git Bash for Windows” environment, so the PATH change should be restricted to uses of latexdiff.

我看不懂Windows安装脚本，里面有关于获取权限的代码，不知是否安全，可以只复制最后一句代码，然后用管理员模式运行。

如果使用Git的过程中发现出现异常，则把Perl的环境变量移动到最底下。

git-latexdiff使用的注意事项：

  1. git-latexdiff不需要提供两份（新旧）对比的工程文件，只需要指定Git的版本标签，使得可以跟Git无缝衔接；
  2. 记得指定输出临时目录--tmpdirprefix，不然输出结果很难找；
  3. --cleanup设置为none，用于保留输出结果；
  4. 指定-b参数可以顺带走完文献引用流程；
